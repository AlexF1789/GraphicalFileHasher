# GraphicalFileHasher
Graphical file hasher for duplicate detection and removal

The core of the program is based on the **fileHasher** repo, this is a graphical reproposal written by using Avalonia for the GUI and a multithreaded approach.

The software is still on **development** so it's strongly advised not to use it on user's sensitive or important files. Please use it only on test scenarios and performing backups if necessary.
